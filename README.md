# AdNabu Test Automation

## Setup and Run Tests

Follow the steps below to execute the automation script and view the report.

### Step 1: Run the Python Script

Open terminal in the project folder and run:

python test_ecommerce.py

This command will execute the test cases and generate the test report.

### Step 2: View the Test Report

After execution, open the generated report using:

Invoke-Item reports\report.html

This will open the HTML report in your default browser where you can review the test results.
